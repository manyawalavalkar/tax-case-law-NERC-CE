# tax-case-law-NERC-CE
This repository contains the code used to conduct the experiments and results presented in the "Named Entity Recognition and Classification and Concept Extraction in the Legal and Tax Domain" research master thesis.

The code is divided into the following categories:

Datasets:
approx_domain_data.ipynb
ibfd_test_set.ipynb
statistics.ipynb
llm_annotator.ipynb

LLM:
zero_shot.ipynb
few_shot.ipynb

Encoders:
-RoBERTa
-LegalBERT
-GLiNER

Evaluation and Error Analysis:
overall_eval.ipynb
